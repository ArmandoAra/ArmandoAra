## 🌟 Featured Project: Spendiary 🪙



**Spendiary** is my flagship project: a 100% offline personal finance app built with React Native 0.81+. 
* **Key Achievement:** Optimized data persistence using **MMKV**, achieving 10x faster speeds than standard AsyncStorage.
* **Architecture:** Implemented **Zustand** for state management and **Shopify FlashList** for high-performance rendering of thousands of records.
* **Privacy:** Built with a "Privacy-by-Design" approach, ensuring data never leaves the device.
