## 🌟 Featured Project: Spendiary 🪙

**Spendiary** is a 100% offline personal finance app built with React Native 0.81+.

- **Performance:** Optimized data persistence with **MMKV**, achieving 10x faster read/write speeds than AsyncStorage
- **Architecture:** **Zustand** for state management + **Shopify FlashList** for high-performance rendering of large datasets
- **Privacy-first:** Built with a Privacy-by-Design approach — data never leaves the device

👉 [View Repository](https://github.com/ArmandoAra/spendiary)
